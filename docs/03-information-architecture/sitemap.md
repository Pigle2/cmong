# 사이트맵 및 정보 구조

## 1. 전체 사이트맵

```
크몽 클론 플랫폼
│
├── 🏠 메인 (/)

│   │   ├── 법률 상담

│   └── 계정 설정 (/buyer/settings)
│       ├── 프로필 수정

## 2. 서비스 카테고리 체계 (상세)

### 2.1 카테고리 구조 (3단계)

| 대분류 | 중분류 | 소분류 (예시) |
|--------|--------|---------------|
| **디자인** | 로고·브랜드 디자인 | 로고 디자인, 브랜드 아이덴티티, 명함, CI/BI |
| | 웹·모바일 디자인 | 웹사이트, 앱 UI/UX, 랜딩페이지, 배너 |
| | 인쇄물·패키지 | 전단지, 포스터, 브로슈어, 패키지 |
| | 일러스트·캐릭터 | 일러스트, 캐릭터, 이모티콘, 웹툰 |
| | PPT·인포그래픽 | PPT 디자인, 인포그래픽, 보고서 |
| **IT·프로그래밍** | 웹사이트 개발 | 홈페이지, 쇼핑몰, 워드프레스, 랜딩페이지 |
| | 앱 개발 | iOS, Android, 하이브리드, 유지보수 |
| | 데이터/AI | 데이터 분석, 크롤링, 머신러닝, 챗봇 |
| | 서버/인프라 | AWS, 클라우드, DevOps, DB설계 |
| | 보안 | 보안 진단, 취약점 분석, 모의해킹 |
| **영상·사진·음향** | 영상 편집 | 유튜브, 숏폼, 행사영상, 교육영상 |
| | 모션그래픽 | 인트로, 타이틀, 인포그래픽 영상 |
| | 촬영 | 제품, 인물, 행사, 부동산 |
| | 사진 편집 | 보정, 합성, 누끼 |
| | 음악/효과음 | 작곡, 편곡, 효과음, 성우 |
| **마케팅** | SNS 마케팅 | 인스타그램, 블로그, 유튜브, 틱톡 |
| | SEO/SEM | 검색최적화, 키워드광고, 컨텐츠 SEO |
| | 콘텐츠 마케팅 | 카드뉴스, 블로그글, 뉴스레터 |
| | 광고 운영 | 페이스북, 구글, 네이버 광고 |
| | 바이럴 마케팅 | 체험단, 서포터즈, 인플루언서 |
| **번역·통역** | 영어 | 일반, 기술, 의학, 법률 |
| | 일본어 | 일반, 비즈니스, 기술 |
| | 중국어 | 간체, 번체, 비즈니스 |
| | 기타 언어 | 독일어, 프랑스어, 스페인어, 베트남어 |
| **문서·글쓰기** | 카피라이팅 | 광고 카피, 슬로건, 네이밍 |
| | 블로그/콘텐츠 | 블로그 포스팅, 리뷰, 체험기 |
| | 보도자료 | 보도자료 작성, 배포 |
| | 자기소개서/이력서 | 자소서, 이력서, 포트폴리오 |
| **비즈니스 컨설팅** | 사업계획서 | 사업계획서, IR자료, 투자유치 |
| | 세무/회계 | 기장대행, 세무신고, 경영분석 |
| | 법률 상담 | 계약서 검토, 법률 자문 |
| | 특허/지식재산 | 특허출원, 상표등록, 디자인등록 |
| **주문제작** | 핸드메이드 | 악세서리, 가죽, 도자기 |
| | 3D 프린팅 | 시제품, 피규어, 건축 모형 |
| | 의류/패션 | 의류 제작, 수선, 리폼 |
| | 소품/굿즈 | 스티커, 엽서, 굿즈 제작 |
| **레슨·코칭** | 어학 | 영어, 일본어, 중국어 회화 |
| | 음악 | 피아노, 기타, 보컬 |
| | 운동/헬스 | PT, 요가, 필라테스 |
| | 자기계발 | 독서, 글쓰기, 시간관리 |
| **기타** | - | 위 카테고리에 해당하지 않는 서비스 |

### 3.1 GNB (Global Navigation Bar)

```
┌─────────────────────────────────────────────────────────────────────┐
│ [로고]  카테고리▼  검색 [_______________🔍]  견적요청  메시지  알림  [프로필▼] │
└─────────────────────────────────────────────────────────────────────┘
```

```
[로고] 카테고리▼  검색 [_______________🔍]  견적요청  [로그인] [회원가입]
