# 이용약관

> 본 문서는 프리랜서 마켓플레이스 플랫폼(이하 "서비스")의 이용약관 기획안입니다.
> 실제 서비스 운영 시 법률 전문가의 검토를 거쳐 최종 확정해야 합니다.

---

## 1. 총칙

### 1.1 목적

본 약관은 주식회사 OOO(이하 "회사")이 운영하는 프리랜서 마켓플레이스 서비스의 이용 조건 및 절차, 회사와 회원 간의 권리·의무 및 책임 사항을 규정함을 목적으로 합니다.

### 1.2 정의

| 용어 | 정의 |
|------|------|
| **서비스** | 회사가 운영하는 프리랜서 마켓플레이스 웹사이트 및 모바일 애플리케이션 |
| **회원** | 본 약관에 동의하고 회원가입을 완료한 자 |
| **구매자** | 서비스를 통해 전문 서비스를 구매하는 회원 |
| **판매자** | 서비스를 통해 전문 서비스를 판매하는 회원 (전문가, 프리랜서) |
| **서비스(긱)** | 판매자가 플랫폼에 등록하여 판매하는 전문 서비스 상품 |
| **주문** | 구매자가 서비스(긱)를 선택하고 결제하여 성립하는 거래 |
| **에스크로** | 거래 대금을 회사가 안전하게 보관하는 결제 보호 시스템 |
| **정산** | 거래 완료 후 수수료를 차감하고 판매자에게 대금을 지급하는 절차 |
| **포인트** | 서비스 내에서 결제 시 사용 가능한 적립금 |
| **쿠폰** | 서비스 결제 시 할인을 제공하는 전자 할인권 |

### 1.3 약관의 효력

| 항목 | 내용 |
|------|------|
| **게시 및 효력** | 서비스 초기 화면 또는 연결 화면에 게시하여 효력 발생 |
| **약관 변경** | 변경 시 적용일 7일 전 공지. 회원 불리한 변경은 30일 전 개별 통지 |
| **동의 간주** | 변경 공지 후 거부 의사 없이 서비스 이용 시 동의로 간주 |
| **거부 권리** | 회원은 변경 약관에 동의하지 않을 권리가 있으며, 이 경우 탈퇴 가능 |

## 2. 회원

### 2.1 회원가입

| 항목 | 내용 |
|------|------|
| **가입 자격** | 만 14세 이상 (BR-AUTH-05) |
| **가입 방법** | 이메일 가입 또는 소셜 로그인 (카카오, 네이버, 구글, 애플) |
| **필수 동의** | 서비스 이용약관, 개인정보 처리방침 |
| **선택 동의** | 마케팅 수신 동의, 이벤트/혜택 알림 |
| **가입 제한** | 탈퇴 후 30일 이내 동일 이메일 재가입 불가 (BR-AUTH-03) |
| **가입 유형** | 구매자 또는 판매자 (전환 가능) |

### 2.2 회원 탈퇴

| 항목 | 내용 |
|------|------|
| **탈퇴 방법** | 서비스 내 설정에서 탈퇴 신청 |
| **탈퇴 제한** | 진행 중인 거래가 있을 경우 탈퇴 불가 (BR-MY-03) |
| **정산 잔액** | 판매자는 미지급 정산 금액 출금 완료 후 탈퇴 가능 (BR-MY-05) |
| **데이터 보존** | 관련 법령에 따라 일정 기간 보관 후 삭제 (상세: 개인정보처리방침) |
| **재가입 제한** | 탈퇴 후 30일간 동일 정보로 재가입 불가 (BR-MY-04) |

### 2.3 회원의 의무

| 의무 | 내용 |
|------|------|
| **정보 정확성** | 가입 시 정확한 정보 제공, 변경 시 즉시 수정 |
| **계정 관리** | 계정 정보(이메일, 비밀번호)의 관리 책임은 회원에게 있음 |
| **법령 준수** | 서비스 이용 시 관련 법령 및 약관 준수 |
| **타인 권리 존중** | 타인의 지적재산권, 초상권, 명예 등을 침해하지 않을 의무 |
| **금지 행위** | 허위 정보 등록, 부정 거래, 직거래 유도 (BR-MSG-02), 플랫폼 우회 행위 |

### 2.4 회원 제재

| 위반 유형 | 제재 수준 | 기간 |
|-----------|-----------|:----:|
| **경고** | 경미한 약관 위반 | - |
| **서비스 이용 제한** | 주요 기능 이용 제한 | 7~30일 |
| **계정 정지** | 전체 서비스 이용 정지 | 30~90일 |
| **영구 퇴출** | 계정 영구 정지 | 영구 |

| 위반 사례 | 적용 제재 |
|-----------|-----------|
| 허위 리뷰 작성 (BR-REV-06) | 리뷰 삭제 + 포인트 회수 + 경고 |
| 리뷰 대가 금전/혜택 제공 (BR-REV-07) | 서비스 이용 제한 |
| 직거래 유도 (BR-MSG-02) | 경고 → 반복 시 계정 정지 |
| 외부 연락처 교환 (BR-MSG-01) | 경고 |
| 타 플랫폼 유도 (BR-SVC-06) | 서비스 반려 + 경고 |

## 3. 서비스 이용

### 3.1 서비스 범위

| 서비스 항목 | 설명 |
|------------|------|
| **서비스 등록/판매** | 판매자가 전문 서비스를 등록하고 판매 |
| **서비스 검색/구매** | 구매자가 서비스를 탐색하고 구매 |
| **에스크로 결제** | 안전한 대금 보관 및 결제 처리 |
| **실시간 채팅** | 구매자-판매자 간 실시간 커뮤니케이션 |
| **견적 요청/발송** | 맞춤형 서비스 견적 시스템 |
| **리뷰 시스템** | 거래 평가 및 리뷰 작성 |
| **정산 시스템** | 판매 수익 정산 및 출금 |

### 3.2 서비스 이용 제한

| 상황 | 조치 |
|------|------|
| 시스템 점검 | 사전 공지 후 일시 중단 (일요일 02:00~05:00) |
| 천재지변 | 불가항력에 의한 서비스 중단 |
| 설비 장애 | 신속한 복구 후 재개 |
| 약관 위반 | 해당 회원의 서비스 이용 제한 |

### 3.3 서비스 중단 고지

| 유형 | 고지 방법 | 고지 시기 |
|------|-----------|-----------|
| **계획된 점검** | 공지사항 + 이메일 + 인앱 알림 | 7일 전 |
| **긴급 점검** | 인앱 알림 + 공지사항 | 가능한 한 빠르게 |
| **서비스 종료** | 개별 통지 + 공지사항 | 30일 전 |

## 4. 거래

### 4.1 거래 성립

| 단계 | 설명 |
|------|------|
| 1 | 구매자가 서비스(긱) 선택 및 주문서 작성 |
| 2 | 결제 완료 시 에스크로 보관 시작 |
| 3 | 판매자 주문 수락 시 거래 성립 (24시간 이내, BR-ORD-01) |
| 4 | 판매자 거절 또는 미수락 시 전액 환불 |

### 4.2 에스크로 결제

| 항목 | 내용 |
|------|------|
| **대금 보관** | 구매자 결제 금액은 회사가 에스크로로 보관 (BR-PAY-01) |
| **보관 기간** | 결제 시점부터 구매 확정 시까지 |
| **지급 조건** | 구매자의 구매 확정 또는 자동 확정 (납품 후 5일, BR-ORD-03) |
| **분쟁 시** | 분쟁 해결 시까지 대금 보류 (BR-ORD-06) |

### 4.3 취소/환불 정책

| 시점 | 구매자 취소 | 판매자 취소 | 환불 금액 |
|------|:---------:|:---------:|-----------|
| **결제 후 ~ 수락 전** | 가능 | 가능 | 전액 환불 |
| **수락 후 ~ 작업 시작 전** | 가능 | 가능 (패널티) | 전액 환불 |
| **작업 시작 후 ~ 중간 납품 전** | 협의 필요 | 가능 (패널티) | 50% 환불 (협의) |
| **중간 납품 이후** | 분쟁 조정 | 불가 | 분쟁 결과에 따름 |
| **납기 초과 3일 이상** | - | - | 전액 환불 (구매자 무조건 취소 가능) |
| **판매자 귀책 사유** | - | - | 전액 환불 |

**판매자 귀책 패널티:**
- 주문 거절/취소 1건당 수락률 감소 (BR-ORD-04: 70% 미만 시 노출 페널티)
- 경고 누적 3회: 서비스 노출 제한 7일
- 경고 누적 5회: 판매 자격 정지 30일

### 4.4 분쟁 해결

```
1. 이의 신청 (납품 후 5일 이내)
   └── 구매자: 사유 + 증빙 자료 제출

2. 판매자 소명 (3일 이내)
   └── 판매자: 반박 의견 + 증빙 자료 제출

3. 운영팀 중재 (7일 이내)
   ├── 구매자 유리: 전액/부분 환불
   ├── 판매자 유리: 거래 확정
   └── 합의 유도: 부분 환불 + 추가 작업 등

4. 결정 통보
   └── 양측에 결정 사유 및 결과 통보
```

## 5. 결제/정산

### 5.1 수수료 구조

| 수수료 항목 | 요율 | 부과 대상 | 설명 |
|------------|:----:|:---------:|------|
| **서비스 수수료** | 20% | 판매자 | 기본 수수료율 20% (마스터 등급: 17%) |
| **결제 수수료** | 1.5%~3.5% | 구매자 | PG 결제 수수료 (결제수단에 따라 1.5%~3.5%) |
| **긴급 의뢰 수수료** | +10% | 구매자 | 24시간 이내 긴급 매칭 |
| **빠른 정산 수수료** | 1.5% | 판매자 | 프로 등급 이상 이용 가능 (BR-PAY-04) |

### 5.2 정산 주기

| 구분 | 정산 시점 | 조건 | 추가 수수료 |
|------|-----------|------|:----------:|
| **일반 정산** | 구매 확정 후 D+7 영업일 | 최소 정산 금액 10,000원 | 없음 |
| **빠른 정산** | 구매 확정 후 1영업일 (D+1) | 최소 정산 금액 10,000원 | 1.5% |
| **자동 정산** | 설정 주기에 따라 자동 | 자동 정산 설정 활성화 | 없음 |

### 5.3 정산 금액 계산

```
서비스 금액 (판매 가격)
    │
    ├── 플랫폼 수수료 (기본 20%, 마스터 17%) → 회사
    │
    └── 판매자 수익 (80~83%)
        │
        ├── 개인 판매자: 원천징수 3.3% (소득세 3% + 지방소득세 0.3%) 차감
        └── 사업자 판매자: 세금계산서 발행 후 전액 지급
```

> ※ 마스터 등급 판매자는 수수료 17% 적용 (판매자 수익 83%)

### 5.4 세금 처리

| 판매자 유형 | 세금 처리 | 증빙 서류 |
|------------|-----------|-----------|
| **개인** | 원천징수 3.3% | 원천징수영수증 (매년 2월 발행) |
| **사업자** | 세금계산서 발행 | 매월 말일 기준 익월 10일까지 |

### 5.5 정산 한도

| 판매자 유형 | 월 정산 한도 | 비고 |
|------------|:----------:|------|
| **개인** | 1,000만원 | BR-PAY-06 |
| **사업자** | 무제한 | BR-PAY-06 |

## 6. 지식재산권

### 6.1 저작권

| 항목 | 권리 귀속 |
|------|-----------|
| **플랫폼 콘텐츠** | 서비스 UI, 디자인, 소프트웨어 등은 회사에 귀속 |
| **판매자 등록 콘텐츠** | 서비스 설명, 포트폴리오, 샘플 등의 저작권은 판매자에 귀속 |
| **납품물** | 거래 완료 후 납품물의 저작권은 계약 조건에 따름 (기본: 구매자에게 이용 허락) |
| **리뷰 콘텐츠** | 리뷰 작성자에 귀속, 플랫폼 내 게시 및 홍보 목적 사용 허락 |

### 6.2 라이선스 범위

| 유형 | 범위 | 설명 |
|------|------|------|
| **기본 라이선스** | 상업적 사용 가능 | 별도 명시 없으면 기본 적용 |
| **확장 라이선스** | 재판매, 대량 사용 가능 | 판매자가 옵션으로 설정 |
| **소스 파일 제공** | 원본 파일 포함 | 패키지 옵션에 따라 |

### 6.3 콘텐츠 게시 규정

| 금지 사항 | 설명 |
|-----------|------|
| 타인 저작물 무단 사용 | 저작권자 허락 없는 이미지, 텍스트, 코드 등 |
| 불법 콘텐츠 | 음란물, 도박, 사행성 콘텐츠 |
| 허위/과장 광고 | 실제와 다른 포트폴리오, 경력 허위 기재 |
| 개인정보 노출 | 타인의 연락처, 실명 등 무단 게시 (BR-SVC-05) |

## 7. 책임 제한

### 7.1 회사의 면책

| 상황 | 면책 범위 |
|------|-----------|
| **거래 당사자 간 분쟁** | 회사는 중개자로서 거래의 당사자가 아니며, 구매자-판매자 간 분쟁에 대한 직접적 책임 없음. 단, 분쟁 조정 서비스 제공 |
| **서비스 품질** | 판매자가 제공하는 서비스의 품질, 적합성, 적시성에 대한 보증 없음 |
| **불가항력** | 천재지변, 전쟁, 테러, 정부 규제 등 불가항력으로 인한 서비스 중단 |
| **제3자 서비스** | PG사, 클라우드 등 제3자 서비스 장애로 인한 서비스 중단 |
| **회원 귀책** | 회원의 계정 관리 소홀, 허위 정보 제공 등으로 인한 손해 |

### 7.2 회사의 책임

| 상황 | 책임 범위 |
|------|-----------|
| **에스크로 보관금** | 에스크로로 보관 중인 대금의 안전한 관리 책임 |
| **개인정보 보호** | 회원 개인정보의 안전한 관리 및 보호 (개인정보처리방침 준수) |
| **서비스 안정성** | 합리적인 수준의 서비스 가용성 유지 노력 |
| **정산 의무** | 거래 완료 건에 대한 적시 정산 |

### 7.3 분쟁 해결

| 단계 | 방법 |
|------|------|
| **1차** | 고객센터를 통한 상담 및 조정 |
| **2차** | 운영팀 분쟁 중재 (§4.4 절차) |
| **3차** | 한국소비자원 또는 관련 분쟁 조정 기관 |
| **4차** | 관할 법원 소송 |

### 7.4 손해배상

| 항목 | 내용 |
|------|------|
| **손해배상 한도** | 회사의 고의 또는 중대한 과실이 아닌 경우, 해당 거래 금액을 한도로 배상 |
| **간접 손해** | 일실이익, 기대이익 등 간접 손해에 대해서는 배상 책임 없음 |
| **면책 조건** | 회원이 약관 위반 또는 불법 행위로 인한 손해는 면책 |
