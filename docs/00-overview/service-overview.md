# 서비스 개요

## 1. 서비스 비전

> "누구나 자신의 전문성으로 가치를 만들고, 누구나 필요한 전문 서비스를 쉽게 찾을 수 있는 대한민국 No.1 프리랜서 마켓플레이스"

### 미션
- 전문가와 의뢰인을 가장 효율적으로 연결하여 양질의 서비스 거래를 실현한다
- 프리랜서가 안정적으로 수익을 창출할 수 있는 생태계를 구축한다
- 신뢰할 수 있는 에스크로 기반 결제 시스템으로 안전한 거래를 보장한다

### 핵심 가치
| 가치 | 설명 |
|------|------|
| **신뢰** | 에스크로 결제, 실명 인증, 리뷰 시스템을 통한 거래 안전성 보장 |
| **전문성** | 체계적인 카테고리와 포트폴리오 시스템으로 전문가 역량 검증 |
| **편의성** | 직관적 UX와 실시간 채팅으로 원활한 소통과 거래 지원 |
| **공정성** | 투명한 수수료 정책과 공정한 분쟁 해결 시스템 |

---

## 2. 서비스 컨셉

### 플랫폼 유형
- **서비스 마켓플레이스**: 판매자(전문가)가 서비스(긱)를 등록하고, 구매자(의뢰인)가 탐색 후 구매하는 C2C/B2C 모델
- **에스크로 기반 거래**: 구매자가 결제하면 플랫폼이 대금을 보관하고, 작업 완료 후 판매자에게 정산

### 타겟 플랫폼
| 플랫폼 | 상세 |
|--------|------|
| **웹 (반응형)** | 데스크톱 + 태블릿 + 모바일 브라우저 대응 |
| **iOS 앱** | iPhone / iPad 네이티브 앱 |
| **Android 앱** | 스마트폰 / 태블릿 네이티브 앱 |

### 차별화 포인트
1. **AI 매칭 시스템**: 의뢰 내용을 분석하여 최적의 전문가를 추천
2. **실시간 협업 도구**: 채팅 내 파일 공유, 작업 진행률 트래킹 기능
3. **전문가 등급 시스템**: 실적 기반 자동 등급 부여 (신규 → 일반 → 프로 → 마스터)
4. **프로젝트 관리 기능**: 마일스톤 기반 진행 관리, 자동 알림

## 3. 비즈니스 모델

### 3.1 수수료 구조

| 항목 | 요율 | 부과 대상 | 설명 |
|------|------|-----------|------|
| **서비스 수수료** | 20% | 판매자 | 거래 완료 시 판매 금액의 20%를 플랫폼 수수료로 차감 |
| **결제 수수료** | 1.5%~3.5% | 구매자 | PG 결제 수수료 (결제수단에 따라 상이) |
| **긴급 의뢰 수수료** | +10% | 구매자 | 24시간 이내 긴급 매칭 요청 시 추가 수수료 |

> ※ 신용카드 3.5%, 간편결제(카카오/네이버/토스) 3.3%, 계좌이체 1.5%, 가상계좌 건당 300원, 휴대폰 3.5%

### 3.2 부가 수익 모델

| 수익원 | 설명 |
|--------|------|
| **프리미엄 광고** | 검색 결과 상위 노출 (CPC 또는 월정액) |
| **프로 뱃지 구독** | 판매자 프로필 강조 표시, 우선 매칭 (월 29,900원) |
| **프리미엄 서비스** | 전담 매니저 배정, 계약서 자동 생성 등 |
| **교육/인증** | 플랫폼 자체 인증 프로그램 운영 |

### 3.3 정산 주기

| 구분 | 정산 주기 | 조건 |
|------|-----------|------|
| **일반 정산** | 구매 확정 후 7영업일 | 최소 정산 금액: 10,000원 |
| **빠른 정산** | 구매 확정 후 1영업일 | 수수료 1.5% 추가 차감 |
| **자동 구매 확정** | 작업 완료 후 5일 경과 시 자동 확정 | 구매자 이의 신청 없을 경우 |

## 6. 서비스 운영 정책

### 6.1 회원 등급 체계 (판매자)

| 등급 | 조건 | 혜택 |
|------|------|------|
| **신규** | 가입 직후 | 기본 서비스 등록 (최대 5개) |
| **일반** | 거래 10건 이상, 평점 4.0 이상 | 서비스 등록 최대 15개, 프로필 배지 |
| **프로** | 거래 50건 이상, 평점 4.5 이상, 응답률 90% | 서비스 등록 무제한, 검색 우선 노출 |
| **마스터** | 거래 200건 이상, 평점 4.8 이상, 운영자 심사 통과 | 전용 배지, 메인 추천, 수수료 15% 할인 |

### 6.2 분쟁 해결 프로세스
1. 구매자 이의 신청 (작업 완료 후 5일 이내)
2. 판매자 소명 기회 부여 (3일 이내)
3. 운영팀 중재 및 결정 (7일 이내)
4. 결정에 따른 환불/부분환불/거래확정 처리

### 6.3 취소/환불 정책

| 시점 | 정책 |
|------|------|
| 주문 수락 전 | 전액 환불 |
| 작업 시작 전 | 전액 환불 |
| 작업 시작 후 ~ 중간 납품 전 | 50% 환불 (협의 가능) |
| 중간 납품 이후 | 환불 불가 (분쟁 조정 가능) |
| 납기 초과 3일 이상 | 구매자 전액 환불 가능 |
| 판매자 귀책 사유 | 전액 환불 |

> ※ 빠른 정산은 프로 등급 이상 판매자만 이용 가능 (수수료 1.5% 추가)
