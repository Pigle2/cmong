| 항목 | 내용 |
|------|------|
| **이름** | 김민준 |
| **나이** | 34세 |
| **직업** | 스타트업 대표 (IT 서비스) |
| **소득** | 연 5,000만 원 |
| **디지털 리터러시** | 높음 |

#### 프로필
- 3년 차 스타트업 대표, 팀원 5명의 소규모 회사 운영
- 디자인, 마케팅 등 비핵심 업무를 외주로 처리해야 하는 상황
- 빠른 실행력과 비용 효율성을 중시

#### 니즈
- 빠르게 적합한 전문가를 찾고 싶음
- 포트폴리오와 리뷰를 통해 실력을 사전 검증하고 싶음
- 반복 의뢰 시 같은 전문가에게 빠르게 재주문하고 싶음
- 프로젝트 진행 상황을 실시간으로 확인하고 싶음

#### 페인 포인트
- 기존 플랫폼에서 전문가 실력 편차가 큼
- 커뮤니케이션이 원활하지 않은 경우가 많음
- 납기 지연이 빈번함
- 수정 요청 범위에 대한 분쟁

#### 사용 시나리오
> "새 앱 런칭을 위해 로고 디자인과 앱 UI를 외주 맡기고 싶어요. 포트폴리오를 보고 실력 있는 디자이너를 빠르게 찾고, 작업 진행 상황을 실시간으로 추적하고 싶어요."

| **소득** | 연 3,500만 원 |
| **디지털 리터러시** | 높음 |

#### 프로필
- 혼자서 마케팅 전반을 담당하는 1인 마케터
- 콘텐츠 제작, 영상 편집, 번역 등 다양한 외주가 필요
- 예산이 제한적이라 합리적인 가격을 선호

#### 니즈
- 가격 비교가 쉬우면 좋겠음
- 소량/반복 주문이 가능한 전문가를 찾고 싶음
- 기업 카드/세금계산서 발행이 필요
- 긴급 의뢰에 빠르게 대응해주는 전문가 필요

#### 페인 포인트
- 매번 새로운 전문가를 찾는 데 시간 소요
- 저가 서비스의 품질이 기대 이하인 경우 다수
- 세금계산서 발행이 안 되는 경우가 있음

#### 사용 시나리오
> "이번 주까지 인스타그램 카드뉴스 10장을 급하게 만들어야 해요. 가격도 합리적이면서 디자인 감각이 좋은 분을 바로 찾고 싶어요."

### 페르소나 4: 프리랜서 디자이너 최유진

| 항목 | 내용 |
|------|------|
| **이름** | 최유진 |
| **나이** | 31세 |
| **직업** | 프리랜서 그래픽 디자이너 (경력 6년) |
| **소득** | 월 300~500만 원 (변동) |
| **디지털 리터러시** | 매우 높음 |

#### 프로필
- 에이전시 퇴사 후 프리랜서 2년 차
- 로고, 브랜딩, 패키지 디자인 전문
- 여러 프리랜서 플랫폼을 병행 사용 중

#### 니즈
- 안정적인 의뢰 유입
- 포트폴리오를 효과적으로 보여주고 싶음
- 수수료가 낮으면 좋겠음
- 클라이언트와의 원활한 소통 도구
- 정산이 빠르고 투명하면 좋겠음

#### 페인 포인트
- 수수료 20%가 부담됨
- 무리한 수정 요청에 시달림
- 저가 경쟁에 노출됨
- 초반 리뷰 확보가 어려움

#### 사용 시나리오
> "포트폴리오를 깔끔하게 정리해서 올리고, 꾸준히 의뢰를 받고 싶어요. 작업 범위를 명확하게 설정해서 추가 수정 분쟁을 줄이고 싶어요."

| **디지털 리터러시** | 매우 높음 |

#### 프로필
- 본업 외 주말/저녁 시간을 활용하여 부업
- 간단한 웹사이트 제작, 크롤링, 자동화 스크립트 등 수행
- 시간 관리가 중요하여 일정 조율이 핵심

#### 니즈
- 일정에 맞게 선택적으로 의뢰를 수주하고 싶음
- 의뢰 범위가 명확하게 정의된 프로젝트 선호
- 간편한 정산 시스템

#### 페인 포인트
- 의뢰인의 요구사항이 모호한 경우가 많음
- 본업과 병행하기 어려운 일정 요구
- 소규모 프로젝트의 커뮤니케이션 비용이 큼

#### 사용 시나리오
> "주말에 여유가 있을 때 간단한 웹사이트 제작 의뢰를 받고 싶어요. 요구사항이 명확하고 일정이 여유있는 프로젝트를 선별해서 수주하고 싶어요."

- 저가 경쟁(AI 번역 등)에 대한 위기감

#### 사용 시나리오
> "의학 논문 전문 번역 서비스를 등록하고 싶어요. 관련 인증서와 경력을 상세하게 올려서 전문성을 증명하고, 적정 단가를 받고 싶어요."
