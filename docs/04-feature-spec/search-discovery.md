# 검색 및 탐색 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 검색 및 탐색 |
| **목적** | 사용자가 원하는 서비스를 효율적으로 찾을 수 있도록 지원 |
| **사용자** | 구매자 (비회원 포함) |

---

## 2. 검색 기능

### 2.1 검색 유형

| 유형 | 설명 |
|------|------|
| **키워드 검색** | 검색창에 키워드 입력하여 서비스 검색 |
| **카테고리 검색** | 카테고리 계층 구조를 탐색하여 서비스 검색 |
| **태그 검색** | 서비스 태그 클릭을 통한 검색 |
| **자동완성** | 검색어 입력 시 실시간 검색어 추천 |

### 2.2 검색 대상 필드

| 대상 | 가중치 | 설명 |
|------|:------:|------|
| 서비스 제목 | 5 | 제목과의 일치도 최우선 |
| 서비스 태그 | 4 | 등록된 태그와의 일치 |
| 서비스 소개 (짧은) | 3 | 짧은 소개 텍스트 |
| 서비스 상세 설명 | 2 | 상세 설명 본문 |
| 판매자 닉네임 | 1 | 판매자 이름/닉네임 |
| 카테고리명 | 3 | 카테고리 이름과의 일치 |

### 2.3 자동완성

```
사용자 입력: "로고 디"
┌────────────────────────┐
│ 🔍 로고 디자인          │
│ 🔍 로고 디자인 의뢰     │
│ 🔍 로고 디자이너        │
│ 📂 디자인 > 로고·브랜드  │ ← 카테고리 바로가기
│ ──────────────────────  │
│ 🕐 최근 검색어           │
│    로고 제작             │
│    명함 디자인           │
└────────────────────────┘
```

- 자동완성 API 응답시간: 200ms 이내
- 최소 2글자 이상 입력 시 동작
- 최대 5개 키워드 + 2개 카테고리 + 3개 최근 검색어

## 3. 필터링

### 3.1 필터 옵션

| 필터 | 타입 | 옵션 |
|------|------|------|
| **카테고리** | 계층 선택 | 대분류 > 중분류 > 소분류 |
| **가격 범위** | 범위 슬라이더 | 최소 ~ 최대 (만 원 단위) |
| **작업일** | 체크박스 | 1일, 3일 이내, 7일 이내, 14일 이내, 30일 이내 |
| **판매자 등급** | 체크박스 | 전체, 프로, 마스터 |
| **평점** | 라디오 | 전체, 4.0+, 4.5+, 4.8+ |
| **수정 횟수** | 체크박스 | 무제한, 3회 이상, 5회 이상 |
| **거래 건수** | 체크박스 | 10건+, 50건+, 100건+ |

### 3.2 정렬 옵션

| 정렬 | 설명 | 기본값 |
|------|------|:------:|
| **추천순** | 관련성 + 인기도 + 최신성 종합 | O |
| **최신순** | 등록일 기준 최신 |  |
| **평점순** | 평균 평점 높은 순 |  |
| **거래순** | 누적 거래 건수 높은 순 |  |
| **가격 낮은순** | Standard 패키지 가격 오름차순 |  |
| **가격 높은순** | Standard 패키지 가격 내림차순 |  |

## 4. 검색 결과 페이지

### 4.1 결과 표시 형식

```
검색결과: "로고 디자인" (총 1,234건)

┌────────────────────────────────────────────┐
│ [필터]  정렬: 추천순 ▼   뷰: ▦ 그리드 / ≡ 리스트 │
├────────────────────────────────────────────┤
│                                            │
│ ┌─────────┐  ┌─────────┐  ┌─────────┐    │
│ │ [이미지] │  │ [이미지] │  │ [이미지] │    │
│ │         │  │         │  │         │    │
│ │ 제목     │  │ 제목     │  │ 제목     │    │
│ │ ★4.9(123)│  │ ★4.8(89)│  │ ★4.7(56)│    │
│ │ 50,000원~│  │ 80,000원~│  │ 30,000원~│    │
│ │ [판매자] │  │ [판매자] │  │ [판매자] │    │
│ └─────────┘  └─────────┘  └─────────┘    │
│                                            │
│         [1] [2] [3] ... [다음 ▶]           │
└────────────────────────────────────────────┘
```

### 4.2 서비스 카드 구성요소

| 요소 | 설명 |
|------|------|
| 대표 이미지 | 서비스 대표 이미지 (썸네일) |
| 서비스 제목 | 최대 2줄 표시 (말줄임) |
| 평점 + 리뷰 수 | ★ 4.9 (123) |
| 시작 가격 | Standard 패키지 가격 |
| 판매자 정보 | 프로필 이미지 + 닉네임 + 등급 배지 |
| 찜 아이콘 | 하트 아이콘 (로그인 사용자) |
| 작업일 | "3일 이내" 등 |

### 4.3 페이징

| 항목 | 값 |
|------|-----|
| 페이지당 표시 건수 | 20건 (그리드), 10건 (리스트) |
| 페이징 방식 | 페이지 번호 (웹), 무한 스크롤 (모바일 앱) |
| 최대 표시 페이지 | 100페이지 (2,000건) |

## 5. 탐색 기능

### 5.1 메인 페이지 섹션

| 섹션 | 내용 | 로직 |
|------|------|------|
| 히어로 배너 | 프로모션/이벤트 배너 슬라이더 | 관리자 설정 |
| 인기 카테고리 | 인기 카테고리 아이콘 그리드 | 거래량 기준 |
| 추천 서비스 | AI 기반 맞춤 추천 (로그인 시) | 사용자 행동 기반 |
| 인기 서비스 | 주간 인기 서비스 랭킹 | 주문수 + 조회수 기반 |
| 신규 서비스 | 최근 등록된 서비스 | 등록일 기준 최신순 |
| 인기 전문가 | 주간 인기 전문가 | 평점 + 거래건수 기반 |

### 5.2 AI 추천 로직

| 요소 | 가중치 | 설명 |
|------|:------:|------|
| 사용자 검색 이력 | 30% | 최근 검색한 키워드/카테고리 |
| 조회 이력 | 25% | 최근 조회한 서비스의 카테고리/태그 |
| 구매 이력 | 25% | 과거 구매한 서비스 기반 |
| 유사 사용자 행동 | 15% | 협업 필터링 기반 |
| 서비스 인기도 | 5% | 전체 인기 보정 |

## 6. 비즈니스 룰

| # | 규칙 |
|---|------|
| BR-SRCH-01 | 판매중지 상태의 서비스는 검색 결과에서 제외 |
| BR-SRCH-02 | 신고가 3건 이상인 서비스는 검색 결과 하위 노출 |
| BR-SRCH-03 | 프로/마스터 등급 판매자의 서비스는 검색 결과 상위 보정 |
| BR-SRCH-04 | 광고 서비스는 검색 결과 상단에 "광고" 라벨과 함께 표시 (최대 3건) |
| BR-SRCH-05 | 검색 결과 0건인 경우 유사 키워드 추천 표시 |
| BR-SRCH-06 | 비로그인 사용자의 추천은 전체 인기 기반 |
| BR-SRCH-07 | 검색 키워드는 익명화하여 트렌드 분석에 활용 |

## 7. 예외 처리

| 시나리오 | 처리 방안 |
|----------|-----------|
| 검색 결과 없음 | "검색 결과가 없습니다" + 유사 키워드 추천 + 인기 서비스 노출 |
| 검색 서비스 장애 | 캐시된 인기 검색 결과 표시 + 장애 안내 |
| 부적절한 검색어 | 필터링 처리 후 "적절하지 않은 검색어입니다" 안내 |
| 특수문자 입력 | SQL Injection / XSS 방지 처리 |
