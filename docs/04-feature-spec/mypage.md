# 마이페이지 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 마이페이지 (구매자/판매자) |
| **목적** | 사용자 계정, 주문, 서비스 등 개인 정보와 거래 관리 |
| **사용자** | 구매자, 판매자 |

---

## 2. 구매자 마이페이지

### 2.1 대시보드

| 섹션 | 표시 내용 |
|------|-----------|
| 프로필 요약 | 프로필 이미지, 닉네임, 회원 등급 |
| 주문 현황 | 진행 중 주문 수, 완료 주문 수 |
| 찜 목록 | 찜한 서비스 수 |
| 쿠폰/포인트 | 사용 가능 쿠폰 수, 보유 포인트 |
| 최근 주문 | 최근 3건의 주문 목록 |
| 추천 서비스 | AI 기반 맞춤 추천 |

### 2.2 주문 내역

| 기능 | 설명 |
|------|------|
| 주문 목록 | 전체/진행중/완료/취소 탭 구분 |
| 주문 상세 | 주문 정보, 상태, 채팅 바로가기 |
| 주문 필터 | 기간별, 상태별 필터 |
| 주문 검색 | 서비스명, 판매자명 검색 |

#### 주문 내역 카드

```
┌──────────────────────────────────────┐
│ 📋 주문번호: ORD-2026030401          │
│ 상태: 🟢 작업중                      │
├──────────────────────────────────────┤
│ [이미지] 프리미엄 로고 디자인         │
│          판매자: 최유진 (프로)        │
│          패키지: Deluxe              │
│          금액: 100,000원             │
│          주문일: 2026-03-01          │
│          예상 완료: 2026-03-06       │
├──────────────────────────────────────┤
│  [채팅] [주문 상세] [취소 요청]       │
└──────────────────────────────────────┘
```

### 2.3 찜 목록

| 기능 | 설명 |
|------|------|
| 찜한 서비스 | 찜한 서비스 목록 (그리드/리스트) |
| 찜한 전문가 | 팔로우한 전문가 목록 |
| 폴더 관리 | 찜 목록을 폴더로 분류 |
| 가격 변동 알림 | 찜한 서비스 가격 변동 시 알림 |

### 2.4 쿠폰/포인트

| 기능 | 설명 |
|------|------|
| 사용 가능 쿠폰 | 현재 사용 가능한 쿠폰 목록 |
| 사용 완료 쿠폰 | 이미 사용한 쿠폰 이력 |
| 만료 예정 쿠폰 | 7일 이내 만료 쿠폰 별도 표시 |
| 포인트 현황 | 보유 포인트, 적립/사용 내역 |
| 쿠폰 등록 | 쿠폰 코드 직접 입력 등록 |

### 2.5 계정 설정

| 설정 항목 | 설명 |
|-----------|------|
| 프로필 수정 | 프로필 이미지, 닉네임, 소개 |
| 비밀번호 변경 | 현재 비밀번호 확인 후 변경 |
| 연락처 변경 | 휴대폰 번호 변경 (인증 필수) |
| 알림 설정 | 이메일/푸시/SMS 알림 ON/OFF |
| 소셜 계정 연동 | 소셜 계정 연동/해제 |
| 결제 수단 관리 | 등록된 카드/결제 수단 관리 |
| 회원 탈퇴 | 탈퇴 사유 선택 후 탈퇴 처리 |

### 3.2 서비스 관리

| 기능 | 설명 |
|------|------|
| 서비스 목록 | 등록한 서비스 목록 (상태별 필터) |
| 서비스 등록 | 새 서비스 등록 |
| 서비스 수정 | 기존 서비스 정보 수정 |
| 서비스 복제 | 기존 서비스를 복제하여 새 서비스 생성 |
| 판매 중지/재개 | 서비스 노출 ON/OFF |
| 통계 | 서비스별 조회수, 전환율, 주문수 |

### 3.3 주문 관리

| 기능 | 설명 |
|------|------|
| 주문 목록 | 대기/진행중/납품완료/완료/취소 탭 |
| 주문 수락/거절 | 새 주문 수락 또는 거절 |
| 작업 진행 관리 | 마일스톤 업데이트, 납품 처리 |
| 납기 관리 | 납기일 확인, 연장 요청 |

### 3.4 정산 관리

| 기능 | 설명 |
|------|------|
| 수익 현황 | 가용 잔액, 정산 대기, 누적 수익 |
| 정산 내역 | 기간별 정산 상세 내역 |
| 출금 요청 | 가용 잔액 출금 요청 |
| 정산 계좌 | 정산 받을 계좌 관리 |
| 세금계산서 | 세금계산서 발행/조회 |

### 3.5 통계/분석

| 지표 | 설명 |
|------|------|
| 매출 통계 | 일별/주별/월별 매출 그래프 |
| 주문 통계 | 주문 건수, 취소율, 완료율 |
| 서비스 통계 | 서비스별 조회수, 전환율 |
| 리뷰 분석 | 평점 추이, 키워드 분석 |
| 고객 분석 | 신규/재구매 비율, 고객 분포 |

### 3.6 프로필 관리

| 항목 | 설명 |
|------|------|
| 기본 정보 | 프로필 이미지, 닉네임, 소개 |
| 전문 분야 | 전문 분야 태그 |
| 경력 사항 | 경력, 학력, 자격증 |
| 포트폴리오 | 대표 포트폴리오 (서비스와 별도) |
| 인증 정보 | 본인 인증, 사업자 인증 상태 |

## 4. 모드 전환 (구매자 ↔ 판매자)

### 4.1 전환 방식
- GNB 프로필 드롭다운에서 "구매자 모드 / 판매자 모드" 토글
- 전환 시 해당 모드의 대시보드로 이동
- 판매자 전환은 판매자 등록 완료 후 가능

### 4.2 모드별 접근 가능 기능

| 기능 | 구매자 모드 | 판매자 모드 |
|------|:----------:|:----------:|
| 서비스 탐색/검색 | O | O |
| 서비스 주문 | O | X |
| 서비스 등록/관리 | X | O |
| 주문 관리 (구매) | O | X |
| 주문 관리 (판매) | X | O |
| 정산 관리 | X | O |
| 메시지 | O | O |
| 리뷰 작성 | O | X |
| 리뷰 답변 | X | O |

## 5. 비즈니스 룰

| # | 규칙 |
|---|------|
| BR-MY-01 | 닉네임 변경은 30일에 1회만 가능 |
| BR-MY-02 | 프로필 이미지는 본인 사진 또는 적절한 이미지만 허용 |
| BR-MY-03 | 회원 탈퇴 시 진행 중인 거래가 있으면 탈퇴 불가 |
| BR-MY-04 | 탈퇴 후 30일 이내 재가입 불가, 데이터는 90일 후 완전 삭제 |
| BR-MY-05 | 판매자 정산 잔액이 있는 상태에서 탈퇴 시 출금 후 탈퇴 |
