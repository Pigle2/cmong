# 메시지/채팅 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 메시지/채팅 시스템 |
| **목적** | 구매자와 판매자 간 실시간 소통 지원 |
| **사용자** | 구매자, 판매자 |

---

## 2. 채팅 기능

### 2.1 채팅 유형

| 유형 | 설명 | 시작 조건 |
|------|------|-----------|
| **구매 전 문의** | 서비스 구매 전 판매자에게 문의 | 서비스 상세 페이지 "문의하기" |
| **주문 채팅** | 주문 진행 중 구매자-판매자 소통 | 주문 생성 시 자동 생성 |
| **견적 채팅** | 견적 요청 관련 소통 | 견적 요청 시 자동 생성 |

### 2.2 메시지 유형

| 유형 | 설명 | 표시 형식 |
|------|------|-----------|
| **텍스트** | 일반 텍스트 메시지 | 말풍선 |
| **이미지** | 이미지 첨부 (JPG, PNG, GIF) | 썸네일 + 클릭 시 원본 |
| **파일** | 파일 첨부 (PDF, ZIP, AI, PSD 등) | 파일 아이콘 + 파일명 + 크기 |
| **견적서** | 판매자가 보내는 견적서 카드 | 전용 카드 UI |
| **납품물** | 판매자가 보내는 최종 납품 파일 | 전용 카드 UI (다운로드) |
| **시스템** | 주문 상태 변경 등 시스템 메시지 | 중앙 정렬 안내 메시지 |

### 2.3 채팅 기능 상세

| 기능 | 설명 |
|------|------|
| **실시간 메시지** | WebSocket 기반 실시간 전송/수신 |
| **읽음 확인** | 상대방 메시지 읽음 표시 (✓✓) |
| **파일 첨부** | 드래그 앤 드롭 또는 첨부 버튼 |
| **이미지 미리보기** | 이미지 인라인 미리보기 |
| **메시지 검색** | 대화 내 키워드 검색 |
| **알림** | 새 메시지 실시간 알림 (푸시/인앱) |
| **타이핑 표시** | 상대방 타이핑 중 표시 |
| **온라인 상태** | 상대방 온라인/오프라인/최근접속 표시 |

### 2.4 파일 첨부 제한

| 항목 | 제한 |
|------|------|
| 파일 크기 | 단일 파일 최대 100MB |
| 파일 개수 | 메시지당 최대 10개 |
| 허용 포맷 | 이미지: JPG, PNG, GIF, WEBP |
| | 문서: PDF, DOC(X), XLS(X), PPT(X), HWP |
| | 디자인: AI, PSD, XD, SKETCH, FIG |
| | 압축: ZIP, RAR, 7Z |
| | 영상: MP4, MOV (최대 500MB) |
| 금지 포맷 | EXE, BAT, SH, DLL 등 실행 파일 |

## 3. 채팅 목록

### 3.1 채팅 목록 구성

```
┌────────────────────────────────────────┐
│ 💬 메시지                     검색 🔍   │
├────────────────────────────────────────┤
│                                        │
│ ┌────────────────────────────────────┐ │
│ │ 🟢 최유진 (프로)          오후 2:30 │ │
│ │ [로고 디자인] 시안 전달드립니다      │ │
│ │                              ● 2  │ │
│ └────────────────────────────────────┘ │
│                                        │
│ ┌────────────────────────────────────┐ │
│ │ ⚫ 정현우               어제       │ │
│ │ [웹개발] 감사합니다!               │ │
│ └────────────────────────────────────┘ │
│                                        │
│ ┌────────────────────────────────────┐ │
│ │ 🟡 한소영               3일 전     │ │
│ │ [번역] 견적서를 보내드립니다        │ │
│ └────────────────────────────────────┘ │
│                                        │
└────────────────────────────────────────┘
```

### 3.2 채팅 목록 정보

| 요소 | 설명 |
|------|------|
| 프로필 이미지 | 상대방 프로필 사진 |
| 온라인 상태 | 🟢 온라인, 🟡 최근 접속, ⚫ 오프라인 |
| 닉네임 + 등급 | 상대방 이름과 판매자 등급 |
| 최근 메시지 | 마지막 메시지 미리보기 (1줄) |
| 시간 | 마지막 메시지 시간 |
| 안 읽은 메시지 | 안 읽은 메시지 개수 뱃지 |
| 연관 서비스 | [서비스명] 표시 |

## 4. 견적서 기능

### 4.1 견적서 작성 필드

| 필드 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| 서비스명 | Select | O | 본인 등록 서비스 중 선택 |
| 견적 금액 | Number | O | 맞춤 견적 금액 |
| 작업 기간 | Number | O | 예상 작업일 |
| 수정 횟수 | Number | O | 포함 수정 횟수 |
| 견적 내용 | Textarea | O | 작업 범위, 포함/미포함 사항 |
| 유효 기간 | Date | O | 견적 유효 기간 (기본 7일) |

### 4.2 견적서 카드 UI

```
┌──────────────────────────────────┐
│ 📋 견적서                        │
├──────────────────────────────────┤
│ 서비스: 로고 디자인               │
│ 금 액: 150,000원                 │
│ 기 간: 5일                       │
│ 수 정: 3회                       │
│ ─────────────────────────────── │
│ 포함사항:                        │
│ · 시안 3개 제공                   │
│ · 소스 파일(AI) 제공              │
│ ─────────────────────────────── │
│ 유효기간: 2026-03-11까지          │
├──────────────────────────────────┤
│    [견적 수락]    [견적 거절]      │
└──────────────────────────────────┘
```

## 5. 비즈니스 룰

| # | 규칙 |
|---|------|
| BR-MSG-01 | 채팅에서 외부 연락처(전화번호, 이메일, SNS) 교환 시도 시 경고 및 필터링 |
| BR-MSG-02 | 결제 전 채팅에서 직거래 유도 감지 시 계정 정지 경고 |
| BR-MSG-03 | 판매자 첫 응답 시간은 프로필에 표시 (평균 응답 시간) |
| BR-MSG-04 | 24시간 이내 미응답 시 구매자에게 안내 + 유사 서비스 추천 |
| BR-MSG-05 | 채팅 내역은 거래 완료 후 1년간 보관, 이후 자동 삭제 |
| BR-MSG-06 | 견적서 유효기간 경과 시 자동 만료 처리 |
| BR-MSG-07 | 악성 메시지(욕설, 비방) 자동 필터링 + 신고 기능 |

## 6. 예외 처리

| 시나리오 | 처리 방안 |
|----------|-----------|
| WebSocket 연결 끊김 | 자동 재연결 (최대 5회), 실패 시 폴링 방식 전환 |
| 파일 전송 실패 | 자동 재전송 3회, 실패 시 안내 메시지 |
| 메시지 순서 불일치 | 서버 타임스탬프 기반 정렬 |
| 동시 접속 | 멀티 디바이스 동기화 지원 |
| 상대방 탈퇴 | "탈퇴한 회원입니다" 표시, 기존 채팅 기록 유지 (읽기 전용) |
