# 알림 시스템 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 알림 시스템 |
| **목적** | 주문/메시지/시스템 이벤트를 실시간으로 사용자에게 전달 |
| **사용자** | 구매자, 판매자 |

---

## 2. 알림 채널

| 채널 | 설명 | 즉시성 |
|------|------|:------:|
| **인앱 알림** | 앱/웹 내 알림 센터 | 실시간 |
| **푸시 알림** | 모바일 앱 푸시 (FCM/APNs) | 실시간 |
| **이메일** | 이메일 알림 | 준실시간 |
| **SMS/카카오 알림톡** | 중요 알림 (결제, 정산 등) | 준실시간 |

## 5. 알림 설정

### 5.1 사용자 설정 항목

| 카테고리 | 설정 | 기본값 |
|----------|------|:------:|
| **주문 관련** | 인앱 / 푸시 / 이메일 / SMS | 전체 ON |
| **메시지** | 인앱 / 푸시 | 전체 ON |
| **마케팅** | 이메일 / 푸시 | OFF |
| **정산** | 인앱 / 이메일 / SMS | 전체 ON |
| **시스템** | 인앱 / 이메일 | 전체 ON |
| **야간 알림 차단** | 22:00~08:00 푸시 차단 | ON |

### 5.2 알림 차단 불가 항목
- 결제/환불 관련 알림 (법적 고지 의무)
- 계정 보안 관련 알림
- 분쟁/제재 관련 알림

## 6. 알림 메시지 템플릿

### 6.1 주요 알림 템플릿

| 이벤트 | 제목 | 내용 |
|--------|------|------|
| 새 주문 | 새 주문이 접수되었습니다 | "{서비스명}" 서비스에 새 주문이 들어왔습니다. 24시간 내 수락해주세요. |
| 납품 완료 | 작업물이 납품되었습니다 | "{서비스명}" 주문의 결과물이 납품되었습니다. 확인해주세요. |
| 정산 완료 | 정산이 완료되었습니다 | {금액}원이 등록된 계좌({은행} {계좌번호 뒤 4자리})로 입금되었습니다. |
| 리뷰 작성 | 새 리뷰가 등록되었습니다 | "{서비스명}" 서비스에 {평점}점 리뷰가 등록되었습니다. |

## 7. 비즈니스 룰

| # | 규칙 |
|---|------|
| BR-NTF-01 | 알림은 최대 90일간 보관 후 자동 삭제 |
| BR-NTF-02 | 읽지 않은 알림은 GNB 배지로 표시 (최대 99+) |
| BR-NTF-03 | 동일 이벤트 반복 알림은 묶어서 표시 ("새 메시지 3건") |
| BR-NTF-04 | 푸시 알림 클릭 시 해당 상세 화면으로 딥링크 |
| BR-NTF-05 | SMS 발송은 하루 최대 10건으로 제한 |
| BR-NTF-06 | 카카오 알림톡 발송 실패 시 SMS로 대체 발송 |
