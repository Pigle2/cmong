# 리뷰/평점 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 리뷰 및 평점 시스템 |
| **목적** | 거래 후 서비스 품질을 평가하고 투명한 정보를 제공 |
| **사용자** | 구매자 (리뷰 작성), 판매자 (리뷰 답변), 비회원 (리뷰 조회) |

---

## 2. 리뷰 작성

### 2.1 리뷰 작성 조건

| 조건 | 설명 |
|------|------|
| **작성 가능 시점** | 구매 확정 후 |
| **작성 기한** | 구매 확정 후 30일 이내 |
| **작성 자격** | 해당 서비스를 실제 구매한 구매자 |
| **수정 가능** | 작성 후 7일 이내 1회 수정 가능 |
| **삭제** | 작성자가 직접 삭제 가능, 삭제 후 재작성 불가 |

### 2.2 리뷰 입력 필드

| 필드 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| **종합 평점** | 별점 (1~5) | O | 0.5 단위 선택 |
| **세부 평점** | 별점 (1~5) | O | 아래 항목별 |
| - 작업 품질 | 별점 | O | 결과물의 완성도 |
| - 의사소통 | 별점 | O | 소통 원활성 |
| - 납기 준수 | 별점 | O | 약속 기한 준수 여부 |
| **리뷰 내용** | Textarea | O | 최소 20자, 최대 1,000자 |
| **첨부 이미지** | Image | X | 최대 5장 (결과물 사진 등) |
| **재구매 의향** | Radio | O | 예 / 아니오 |

## 5. 평점 계산

### 5.1 서비스 평균 평점 계산

```
서비스 평균 평점 = Σ(각 리뷰 종합 평점) / 리뷰 수
(소수점 둘째 자리 반올림, 예: 4.82)
```

### 5.2 판매자 전체 평점

```
판매자 평점 = Σ(모든 서비스의 평균 평점 × 해당 서비스 리뷰 수) / 전체 리뷰 수
```

### 5.3 평점 표시 기준

| 리뷰 수 | 표시 |
|---------|------|
| 0건 | "아직 리뷰가 없습니다" |
| 1~4건 | 평점 표시 (단, "리뷰 수가 적어 변동될 수 있습니다" 안내) |
| 5건 이상 | 평점 정상 표시 |

## 6. 리뷰 관리 (관리자)

### 6.1 관리 기능

| 기능 | 설명 |
|------|------|
| 리뷰 숨김 | 부적절한 리뷰 비공개 처리 |
| 리뷰 삭제 | 정책 위반 리뷰 삭제 |
| 신고 리뷰 관리 | 신고된 리뷰 검토 및 처리 |
| 허위 리뷰 감지 | AI 기반 허위 리뷰 자동 감지 |

### 6.2 리뷰 신고 사유

| 사유 | 설명 |
|------|------|
| 허위/조작 리뷰 | 실제 거래와 무관한 허위 리뷰 |
| 비방/욕설 | 부적절한 언어 사용 |
| 개인정보 노출 | 연락처, 실명 등 개인정보 포함 |
| 광고/스팸 | 다른 서비스 홍보 등 |
| 거래와 무관 | 서비스 내용과 무관한 리뷰 |
