# 결제/정산 시스템 기능 명세

## 1. 기능 개요

| 항목 | 내용 |
|------|------|
| **기능명** | 결제 및 정산 시스템 |
| **목적** | 안전한 에스크로 결제와 투명한 정산 시스템 제공 |
| **사용자** | 구매자 (결제), 판매자 (정산), 관리자 (정산 관리) |

## 2. 결제 시스템

### 2.1 결제 수단

| 결제 수단 | PG사 | 수수료율 | 비고 |
|-----------|------|:--------:|------|
| **신용/체크카드** | NHN KCP | 3.5% | 주요 결제 수단 |
| **카카오페이** | 카카오페이 | 3.3% | 간편결제 |
| **네이버페이** | 네이버페이 | 3.3% | 간편결제 |
| **토스페이** | 토스페이먼츠 | 3.3% | 간편결제 |
| **계좌이체** | NHN KCP | 1.5% | 실시간 계좌이체 |
| **가상계좌** | NHN KCP | 건당 300원 | 무통장입금 |
| **휴대폰결제** | 다날 | 3.5% | 소액 결제 시 |

### 2.2 에스크로 프로세스

```
구매자 결제
    │
    ▼
┌─────────────┐
│ 에스크로     │    대금 보관
│ (플랫폼 보관) │◀──────────────────────────────┐
└──────┬──────┘                                │
       │                                       │
       ▼                                       │
  구매 확정?                                    │
  ├── Yes ──▶ 판매자 수익 확정                    │
  │           │                                │
  │           ▼                                │
  │     수수료 차감 (등급별 20%/17%)              │
  │           │                                │
  │           ▼                                │
  │     정산 대기                               │
  │           │                                │
  │           ▼                                │
  │     정산일에 출금                            │
  │                                            │
  ├── 취소 ──▶ 구매자에게 환불 ──────────────────┘
  │
  └── 분쟁 ──▶ 운영팀 중재 후 처리
```

### 2.3 결제 프로세스 상세

```
1. 주문서 작성 완료
2. 결제 수단 선택
3. 결제 금액 확인 (서비스 금액 + 결제 수수료)
4. PG 결제창 호출
5. 결제 인증 (카드사/은행)
6. 결제 승인 처리
7. 에스크로 보관 시작
8. 주문 생성 + 판매자 알림
```

### 2.4 결제 금액 계산

```
서비스 금액    = 패키지 가격 + 추가 옵션
쿠폰 할인      = 정액 or 정률 할인
포인트 사용    = 최대 결제금액의 50%까지
결제 수수료    = (서비스 금액 - 쿠폰 - 포인트) × 결제수단별 수수료율
──────────────────────────────────────────
총 결제 금액   = 서비스 금액 - 쿠폰 - 포인트 + 결제 수수료

※ 수수료율은 결제수단에 따라 상이 (1.5%~3.5%). 아래 예시는 신용카드(3.5%) 기준
```

## 3. 정산 시스템

### 3.1 정산 구조

```
거래 금액 (구매자 결제액)
    │
    ├── 결제 수수료 (결제수단별 상이) → PG사
    │
    └── 서비스 금액 (100%)
        │
        ├── 플랫폼 수수료 (등급별 상이) → 플랫폼
        │   ├── 신규/일반/프로: 20%
        │   └── 마스터: 17% (기본 수수료 15% 할인)
        │
        └── 판매자 수익 (80~83%) → 판매자
            │
            ├── 개인: 원천징수 (3.3%) 차감 후 지급
            └── 사업자: 세금계산서 발행 후 전액 지급
```

### 3.2 등급별 플랫폼 수수료율

| 판매자 등급 | 수수료율 | 비고 |
|------------|:--------:|------|
| **신규** | 20% | 기본 수수료 |
| **일반** | 20% | 기본 수수료 |
| **프로** | 20% | 기본 수수료 |
| **마스터** | 17% | 기본 수수료 15% 할인 |

> ※ 마스터 등급 판매자는 수수료율 17% 적용 (기본 수수료 15% 할인)

### 3.4 정산 주기

| 구분 | 정산 시점 | 조건 | 수수료 |
|------|-----------|------|--------|
| **일반 정산** | 구매 확정 후 7영업일 | 최소 정산 금액: 10,000원 | 없음 |
| **빠른 정산** | 구매 확정 후 1영업일 | 최소 정산 금액: 10,000원 | 1.5% |

### 3.5 정산 처리 프로세스

```
구매 확정
    │
    ▼
정산 대기 (D+0)
    │
    ▼
정산 가능 (D+7 또는 D+1 빠른 정산)
    │
    ├── 자동 정산 (설정 시)
    │   └── 설정된 계좌로 자동 입금
    │
    └── 수동 출금 요청
        └── 요청 후 1~2영업일 내 입금
```

### 3.6 정산 관리 기능 (판매자)

| 기능 | 설명 |
|------|------|
| 정산 내역 조회 | 기간별/상태별 정산 내역 조회 |
| 출금 요청 | 가용 잔액에서 출금 요청 |
| 자동 정산 설정 | 매주/매월 자동 정산 설정 |
| 정산 계좌 관리 | 정산 받을 계좌 등록/변경 |
| 세금계산서 조회 | 월별 세금계산서 발행/조회 |
| 수익 통계 | 월별/분기별 수익 통계 |

## 4. 쿠폰/포인트 시스템

### 4.1 쿠폰 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **정액 할인** | 고정 금액 할인 | 5,000원 할인 |
| **정률 할인** | 결제 금액의 n% 할인 | 10% 할인 (최대 10,000원) |
| **신규 가입** | 신규 회원 환영 쿠폰 | 5,000원 할인 (3만원 이상) |
| **재주문** | 동일 판매자 재주문 쿠폰 | 3% 할인 |
| **이벤트** | 기간 한정 이벤트 쿠폰 | 다양 |

### 4.2 쿠폰 적용 규칙

| # | 규칙 |
|---|------|
| CP-01 | 쿠폰은 주문 1건당 1장만 적용 가능 |
| CP-02 | 쿠폰과 포인트 동시 사용 가능 |
| CP-03 | 정률 쿠폰은 최대 할인 금액 제한 있음 |
| CP-04 | 최소 주문 금액 조건 충족 시만 사용 가능 |
| CP-05 | 쿠폰 유효기간 경과 시 자동 소멸 |
| CP-06 | 환불 시 미사용 쿠폰 복원 (유효기간 내) |

### 4.3 포인트 시스템

| 항목 | 내용 |
|------|------|
| 적립률 | 결제 금액의 1% |
| 사용 조건 | 최소 1,000포인트 이상, 결제금액의 50% 이내 |
| 유효기간 | 적립일로부터 1년 |
| 적립 시점 | 구매 확정 시 |
| 소멸 처리 | 매월 1일 만료 포인트 자동 소멸 |

## 5. 환불 처리

### 5.1 환불 프로세스

```
환불 요청
    │
    ▼
환불 사유 확인
    │
    ├── 자동 환불 (전액)
    │   ├── 판매자 수락 전 취소
    │   └── 판매자 주문 거절
    │
    └── 수동 환불 (부분/전액)
        ├── 구매자-판매자 협의 취소
        └── 운영팀 분쟁 처리 결과
```

### 5.2 환불 소요 기간

| 결제 수단 | 환불 소요 기간 |
|-----------|:-------------:|
| 신용카드 | 3~5영업일 |
| 체크카드 | 3~5영업일 |
| 간편결제 | 1~3영업일 |
| 계좌이체 | 1~3영업일 |
| 가상계좌 | 환불 계좌 입력 후 1~3영업일 |

## 6. 세금 처리

### 6.1 개인 판매자

| 항목 | 내용 |
|------|------|
| 원천징수 | 사업소득 3.3% (소득세 3% + 지방소득세 0.3%) |
| 원천징수영수증 | 매년 2월 발행 |
| 종합소득세 신고 | 5월 개인 신고 (플랫폼 원천징수 내역 참조) |

### 6.2 사업자 판매자

| 항목 | 내용 |
|------|------|
| 세금계산서 | 매월 말일 기준 익월 10일까지 발행 |
| 부가세 | 판매자가 별도 신고/납부 |
| 매출 증빙 | 정산 내역서 다운로드 가능 |

## 8. 예외 처리

| 시나리오 | 처리 방안 |
|----------|-----------|
| PG 결제 실패 | 재시도 안내 + 다른 결제 수단 추천 |
| PG사 장애 | 대체 PG사 자동 전환 (Failover) |
| 이중 결제 | 자동 감지 후 중복분 즉시 환불 |
| 정산 계좌 오류 | 정산 보류 + 계좌 재확인 안내 |
| 환불 지연 | 고객센터 에스컬레이션, 수동 환불 처리 |
