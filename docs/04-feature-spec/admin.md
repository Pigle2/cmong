## 4. 회원 관리

### 4.1 기능 목록

| 기능 | 설명 |
|------|------|
| 회원 목록 | 전체/구매자/판매자 필터, 검색 |
| 회원 상세 | 가입 정보, 활동 이력, 거래 내역 |
| 회원 상태 변경 | 활성/정지/탈퇴 상태 변경 |
| 제재 관리 | 경고, 일시정지(7/30/90일), 영구정지 |
| 판매자 승인 | 판매자 전환 신청 승인/거절 |
| 등급 관리 | 판매자 등급 수동 조정 |

### 4.2 제재 사유

| 사유 | 제재 수준 |
|------|-----------|
| 직거래 유도 | 경고 → 7일 정지 → 영구정지 |
| 허위 리뷰 | 경고 → 30일 정지 |
| 저작권 침해 | 서비스 삭제 + 경고 |
| 욕설/비방 | 경고 → 7일 정지 |
| 연락처 노출 | 경고 (반복 시 정지) |
| 사기 행위 | 즉시 영구정지 + 법적 조치 |

## 11. 비즈니스 룰

| # | 규칙 |
|---|------|
| BR-ADM-01 | 관리자 계정은 2FA(이중 인증) 필수 |
| BR-ADM-02 | 관리자 모든 활동은 감사 로그(Audit Log) 기록 |
| BR-ADM-03 | 민감 작업(제재, 환불, 정산) 시 사유 기록 필수 |
| BR-ADM-04 | 슈퍼관리자 계정은 최소 2개 이상 유지 |
| BR-ADM-05 | 관리자 비밀번호는 90일마다 변경 필수 |
| BR-ADM-06 | 외부 IP에서의 관리자 접근은 VPN 필수 |
