### 1.2 UI 컴포넌트 명세

| 컴포넌트 | 속성 | 설명 |
|----------|------|------|
| 히어로 배너 | 자동 슬라이드 (5초 간격), 좌우 화살표, 인디케이터 | 최대 5장 |
| 카테고리 아이콘 | 56x56px 아이콘 + 텍스트, hover 효과 | 가로 스크롤 (모바일) |
| 서비스 카드 | 280x320px, 그림자 효과, hover 시 확대 | 반응형 그리드 |
| 섹션 헤더 | 제목 + "전체보기" 링크 | 좌/우 정렬 |
