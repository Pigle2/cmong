# 채팅 화면 설계

---

## 1. 채팅방 더보기 메뉴

### 1.1 와이어프레임 (PC)

```
┌──────────────────────────────────────────────────────┐
│ ◀ 최유진 (프로)  🟢 온라인             🔍  📎  ⋮   │
├──────────────────────────────────────────────────────┤
│                                          ┌─────────┐ │
│                                          │ 알림 끄기│ │
│  (채팅 메시지 영역)                       │ 검색     │ │
│                                          │─────────│ │
│                                          │ 나가기   │ │
│                                          └─────────┘ │
│                                                      │
│                                                      │
├──────────────────────────────────────────────────────┤
│ [+ 첨부]  [메시지를 입력하세요...]         [전송 ▶]   │
└──────────────────────────────────────────────────────┘
```

### 1.2 와이어프레임 (모바일)

```
┌─────────────────────┐
│ ◀ 최유진 🟢      ⋮  │
├─────────────────────┤
│              ┌─────┐│
│              │알림  ││
│  (채팅 영역)  │검색  ││
│              │─────││
│              │나가기││
│              └─────┘│
│                     │
│                     │
├─────────────────────┤
│ [+] [메시지...]  [▶]│
└─────────────────────┘
```

### 1.3 UI 컴포넌트 명세

| 컴포넌트 | 속성 | 설명 |
|----------|------|------|
| 더보기(⋮) 버튼 | 아이콘 24x24px, 우측 상단 | 클릭/탭 시 드롭다운 메뉴 오픈 |
| 드롭다운 메뉴 | 최소 너비 120px, 그림자 효과 | 항목: 알림 끄기, 검색, 나가기 |
| 나가기 항목 | 텍스트 색상 `text-destructive` (빨간색) | 구분선으로 일반 항목과 분리 |

## 4. 나가기 확인 다이얼로그

### 4.1 와이어프레임

```
┌──────────────────────────────────┐
│                                  │
│  (반투명 오버레이 배경)            │
│                                  │
│   ┌────────────────────────────┐ │
│   │                            │ │
│   │    채팅방 나가기             │ │
│   │                            │ │
│   │    채팅방을 나가시겠습니까?  │ │
│   │                            │ │
│   │    · 대화 내용은 삭제되지    │ │
│   │      않습니다.              │ │
│   │    · 상대방에게 나가기가     │ │
│   │      알림됩니다.            │ │
│   │    · 상대방이 메시지를       │ │
│   │      보내면 채팅방이 다시    │ │
│   │      나타납니다.            │ │
│   │                            │ │
│   │  [취소]          [나가기]   │ │
│   │                            │ │
│   └────────────────────────────┘ │
│                                  │
└──────────────────────────────────┘
```

### 4.2 UI 컴포넌트 명세

| 컴포넌트 | 속성 | 설명 |
|----------|------|------|
| 다이얼로그 | 최대 너비 400px, 라운드 12px, 중앙 정렬 | AlertDialog 컴포넌트 사용 |
| 제목 | `text-lg font-semibold` | "채팅방 나가기" |
| 설명 | `text-sm text-muted-foreground` | 안내 문구 3줄 |
| 취소 버튼 | `variant="outline"` | 다이얼로그 닫기 |
| 나가기 버튼 | `variant="destructive"` (빨간색 배경, 흰색 텍스트) | 나가기 API 호출 |
| 오버레이 | 반투명 검정 배경 (`bg-black/50`) | 외부 클릭 시 닫기 |

## 8. 화면 전환 흐름 (채팅방 나가기)

```
채팅 목록
 ├──▶ 스와이프 / 우클릭 / 길게 누르기
 │     └──▶ "나가기" 선택
 │           └──▶ 확인 다이얼로그 (섹션 4)
 │                 ├── [취소] ──▶ 다이얼로그 닫기
 │                 └── [나가기] ──▶ API 호출
 │                       ├── 성공 ──▶ 목록에서 제거 + 상대방에게 시스템 메시지
 │                       ├── 실패 (주문) ──▶ 차단 다이얼로그 (섹션 5)
 │                       └── 실패 (분쟁) ──▶ 차단 다이얼로그 (섹션 6)
 │
채팅방 내부
 └──▶ 더보기(⋮) 메뉴
       └──▶ "나가기" 선택
             └──▶ (동일 흐름)
```

### 9.2 UI 컴포넌트 명세

| 컴포넌트 | 속성 | 설명 |

| 구분선 | 좌우 가로선 + 중앙 텍스트 | 시스템 메시지와 동일 스타일 |
